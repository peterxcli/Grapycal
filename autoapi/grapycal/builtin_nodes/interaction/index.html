run(task: Callable[[], None], background=True, to_queue=True, **kwargs)#

Run a task in the nodes context i.e. the stdout and errors will be redirected to the nodes output attribute and be displayed in front-end.

Args:

  • task: The task to run.

  • background: If set to True, the task will be scheduled to run in the background thread. Otherwise, it will be run in the current thread immediately.

  • to_queue: This argument is used only when background is True. If set to True, the task will be pushed to the BackgroundRunners queue. If set to False, the task will be pushed to its stack. See BackgroundRunner for more details.
